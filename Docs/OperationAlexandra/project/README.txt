OPERATION ALEXANDRA
A 64KB videogame for Amstrad CPC



How to exec this source code
----------------------------

1. Open Winape.
2. Press F3.
3. Select File and then Open.
4. Select the file "H Ppal.asm" from this source code.
5. Press F9.
6. Press OK.
